# Jeff BezOS
I thought it would be funny

## Building
1. install `clang`, `nasm`, `make`, `ld`
2. run `make x86-64`
3. kernel will be sitting in the main directory called `bezos.bin`

## Running
1. install `qemu-system-x86_64`
2. run with `qemu-system-x86_64 bezos.bin`
